Literate programming, Notebook programming.

- Paradigme en sciences des données.
- Cf. Knuth 1984 Literate programming, *The Computer Journal* : Idée d’inverser le processus d’écriture du code où écrit ce que souhaite faire, puis élabore le code. 
- Wolfram *Mathematica* 1988 : Très bon logiciel.
- Jupyter Notebooks (Python) 2014 : qui ont réellement popularisé l’idée d’un environnement de programmation de ce genre.

Construire un site. Système qui dès que l’on voit qqch d’intéressant dans Voyant, de le reproduire dans Voyant. = Faciliter la prise en main pour les humanistes.

Dans Voyant, à partir des outils complémentaires, il est possible d’importer des contenus.

Il y a deux composants principaux : des cellules de texte et des cellules de code qui peuvent être exécutées. Tout en haut, il existe également une cellule de métadonnées qui permet d’ajouter des mots-clefs, etc.

Si les cellules sont exécutées dans l’ordre, possible d’accéder à un résultat.
